PigEditor by Romain

Eclipse plugin for Apache Pig

Editor for Pig simliar to what Eclipse can do with Java:

  • automatic error highlighting
  • check alias name existence
  • auto complete keywords, UDF names (built-in, piggybank)
  • outline

It is possible to edit Pig scripts remotely with the Remote System Explorer perspective (tip from @amgadmadkour!)

There is still a lot to improve and there are some false positive errors highlighted (trying to be in sync with Pig 0.9 without Macros for now), but feel free to send me your feedback/scripts/pull requests.
