PigEditor

Eclipse plugin for Apache Pig

Editor for Pig similiar to what Eclipse can do with Java:

  • syntax/errors highlighting
  • check alias name existence
  • auto complete keywords, UDF names
  • outline...

It is possible to edit Pig scripts remotely with the Remote System Explorer perspective (tip from @amgadmadkour!).

There is still a lot to improve and false positive errors highlighted are possible (trying to be in sync with Pig 0.9 without Macros for now), but feel free to send me your feedback/scripts/pull requests.
